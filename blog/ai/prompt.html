AI Prompt 工程与应用

Prompt Design（逐题 Q/A）

Q1：高质量 Prompt 的结构？

A1：明确目标、给足上下文、约束输出格式、定义评价标准、提供失败兜底。结构化 Prompt 能稳定输出质量。

Q2：Few-shot 何时最有效？

A2：当任务边界清晰、样例覆盖关键场景时效果最好。通常 3-8 个高质量样例比大量低质量样例更有效。

Q3：CoT 总是更好吗？

A3：不是。复杂推理题常有收益，但事实问答可能引入冗余和幻觉。应根据任务按需开启，避免无谓成本。

Q4：如何防 Prompt 注入？

A4：把系统指令和用户输入严格隔离，对外部内容做清洗与策略校验。高风险动作不让模型直接执行。

Q5：Prompt 如何产品化管理？

A5：需要版本控制、A/B 实验、回滚机制和质量监控。把 Prompt 当配置资产，而不是散落在业务代码里。

Q6：Prompt 优化怎么量化？

A6：看任务成功率、人工修订率、响应时延、token 成本和用户满意度，形成可回归的指标闭环。
